== Description ==

Simple social network sharing plugin with sticky bar. It's flexible and easily extensible. Configure visual appearance. Choose one of several themes provided. share on whatsapp included.

<strong>Available social networks:</strong>
facebook, twitter, google plus, linked in, pininterest, whatsApp, stumbleupon, email

<strong>Comming Soon</strong>
WordPress, Buffer, Digg, Tumblr, Reddit, Delicious, Evernote, Pocket

Demo : <a href="http://freebiesdownload.com/">http://freebiesdownload.com/</a>

== Installation ==

1. Upload `/pwn-social/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to pwn social setting page under setting menu
